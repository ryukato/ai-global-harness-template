## Planning / Design Profile

This repository is a lightweight planning and design workspace, not an application codebase.

- Prioritize product briefs, requirements, acceptance criteria, design notes, and handoff clarity.
- Do not assume that code, tests, build tools, or deployable runtime files exist.
- Do not create implementation code unless the user explicitly changes the scope.
- Treat Jira, Figma, Confluence, and GitHub access as local developer setup.
- Ask for explicit approval before creating or updating external issues, documents, or design artifacts.
