# complete-task

Use this command when implementation and review are complete.

## Prompt

Update:

```text
.ai-workspace/active/<TASK-ID>/final-summary.md
.ai-workspace/completed/<TASK-ID>.md
```

Verify:

- explorer output exists
- implementer output exists
- reviewer output exists
- architect output exists if required
- acceptance criteria are addressed
- tests/checks are recorded
