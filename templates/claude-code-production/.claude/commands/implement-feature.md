# implement-feature

Use this command when implementing a work item.

Read the selected Jira ticket, GitHub issue, or `work-items/*` file.

Then use:

- `jira-ticket-analysis` if the source is Jira-like
- `github-issue-analysis` if the source is GitHub-like
- `implementation-planning`
- `project-tech-stack`
- `domain-context` if business behavior changes
- `test-strategy`

Implement the task completely and report results using the format in `CLAUDE.md`.
