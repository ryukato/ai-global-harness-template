# AGENTS.md

This file mirrors repository-wide rules for AI coding agents that support AGENTS.md.

Claude Code should use `CLAUDE.md` as the primary instruction file.

## Rules

- Keep changes small and focused.
- Do not modify unrelated files.
- Follow repository architecture and domain documents.
- Implement tasks completely.
- Run relevant tests/checks.
- Report changed files and verification results.

## Completion Standard

Do not mark a task complete if acceptance criteria are not met, implementation is partial, tests were not considered, or unrelated files were changed.
