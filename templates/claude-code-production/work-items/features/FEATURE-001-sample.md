# FEATURE-001 User Search

## Goal

Allow users to search by name.

## Background

The current user list requires manual filtering.

## Requirements

- Search by name.
- Case-insensitive matching.
- Pagination support.

## Acceptance Criteria

- Matching users are returned.
- Empty keyword behavior is defined.
- Pagination works.

## Out of Scope

- Full-text search engine integration.
- UI redesign.
