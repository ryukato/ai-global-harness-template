# Design Handoff Guidelines

## Include

- Figma link and frame names
- Requirement or Jira links
- Flow summary
- UI states
- Responsive behavior
- Copy notes
- Accessibility notes
- Data and permission assumptions
- Known tradeoffs

## Avoid

- Sending screenshots without source links.
- Leaving error or empty states unspecified.
- Mixing confirmed requirements with exploratory ideas.
- Asking engineering to infer business rules from visual layout alone.
