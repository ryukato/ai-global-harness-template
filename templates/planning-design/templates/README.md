# Planning / Design Artifact Templates

Use these templates for product, design, QA, and review deliverables.

## Included

- `planning/release-overview-template.md`
- `planning/requirement-template.md`
- `design/design-handoff-template.md`
- `jira/ticket-template.md`
- `qa/qa-test-plan-template.md`
- `retrospective/sprint-review-retrospective-template.md`

## Not Included

Backend technical notes, frontend technical notes, API review notes, and coding-agent task instructions are development handoff artifacts. Keep them in a development-oriented harness profile unless the planning/design workspace explicitly takes ownership of those deliverables.
